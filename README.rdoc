#8chan Grabber

###Dependencies:

* Python2.7
* BeautifulSoup4

In order to be able to run 8chan Grabber, you'll need to have BeautifulSoup4 installed.
This is easily done with pip or easy_install:

######For Pip:

`pip install beautifulsoup4'`

#####For Easy_install:

`easy_install beautifulsoup4`

###Usage:

Usage is simple, just run `./dl.py` on GNU/Linux or run `python dl.py` which works on most platforms with
python2.7 installed
